Total wine: 750ml Scotch Whiskey under $50

link

I go to the horizontal scroll bar directly for different alcohol genres there and select Scotch. I looked at the marked volume on the products in this section and select a Limited Time Special. I like the part that in addition to the overview of the wine (e.g. brand, country…), there is also a section for Product Highlight with the tags highlighting the keywords in the paragraph so that I don’t have to spend time reading. In the product detail there is also a ‘progress bar’ showing the range of mild-complex and delicate-rich which is great for people with little knowledge of this type of whiskey. Food pairing is shown with pictures and names of food. A table is used to compare similar products.

Summary

For viewing the genre of items, a horizontal header menu is very common and the following popup sub-titles. A scrollable page turner is universal for seeing products in different directions and for more details. There are also filter options and sort function next to the search bar or on the side. A pop up window for shopping cart confirmation and similar products is also very common. In the detail page, separate pages for different content (e.g. overview, size chart) indicated by tabs on the top seem universal. The progress bar in the wine website is very uncommon and can only be used to indicate the level or degree. I was most unsatisfied with the skateboard website since I’m looking for long complete skateboards, but there is only filter for complete ones but no selection for size. In the detail page, I was also expecting to see the details of each part but they only provide pictures of the product in an overall look. I wish there is a close up picture or allow the users to zoom in/out. Some of the website have automatic unit switching (inch to cm) and I think it is a great feature to improve the equity.
